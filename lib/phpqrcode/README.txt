Instructions d'installation de phpqrcode :

1. Téléchargez la bibliothèque phpqrcode depuis SourceForge :
   https://sourceforge.net/projects/phpqrcode/files/latest/download

2. Extrayez le contenu de l'archive ZIP.

3. Copiez tous les fichiers (notamment qrlib.php et les dossiers associés) dans ce dossier (lib/phpqrcode/).

Le fichier qrlib.php doit se trouver exactement ici :
lib/phpqrcode/qrlib.php
