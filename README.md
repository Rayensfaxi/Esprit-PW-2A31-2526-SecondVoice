# SecondVoice

## Overview
SecondVoice is an academic web platform designed to support people who face digital access difficulties. It provides human-centered assistance to help users complete online procedures more easily.

This project was developed at **Esprit School of Engineering – Tunisia** during the academic year **2025–2026**.

## Features
- User management
- Request management
- Appointment management
- Document management
- Complaint management
- AI-assisted support workflows

## Tech Stack
- HTML5
- CSS3
- JavaScript

### Frontend
- Static multi-page interface in `frontend/`
- Shared styles and interactions in `frontend/assets/`

### Backend
- Service pages and supporting assets in `backend/`
- Shared styling and scripts in `backend/assets/`

## Architecture
The repository is organized into two main modules:
- `frontend/`: user-facing interfaces (chatbot, voice, images, community, profile, settings, subscription)
- `backend/`: service/informational pages and base web assets

Both modules use static web technologies and can be served by any HTTP server.

## Getting Started
1. Clone the repository.
2. Open `frontend/index.html` for the user interface.
3. Open `backend/index.html` for backend-side pages.
4. Optionally serve the project with a local HTTP server.
