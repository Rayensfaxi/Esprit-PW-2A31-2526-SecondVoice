# SecondVoice

## Overview
SecondVoice est un projet academique (2025-2026) actuellement en phase **template**.

## Architecture actuelle (template)
Le projet respecte une structure MVC minimale:

- `view/`
  - `frontoffice/` (templates front office)
  - `backoffice/` (templates back office)
- `controller/` (vide pour le moment)
- `model/` (vide pour le moment)

## Structure
```text
Second voice/
  view/
    frontoffice/
    backoffice/
  controller/
  model/
  README.md
```

## Notes
- Aucun backend logique n'est implemente pour l'instant.
- Les dossiers `controller` et `model` sont prepares pour les prochaines etapes.
