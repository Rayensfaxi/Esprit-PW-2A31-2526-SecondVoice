# SecondVoice

## Description
SecondVoice est une plateforme web intelligente conçue pour accompagner les personnes en difficulté avec le numérique. Elle propose une assistance humaine pour aider les utilisateurs à effectuer leurs démarches en ligne facilement.

## Objectifs
- Réduire la fracture numérique
- Faciliter l’accès aux services
- Offrir un accompagnement humain

## Fonctionnalités
- Gestion des utilisateurs
- Gestion des demandes
- Gestion des rendez-vous
- Gestion des documents
- Gestion des réclamations

## Intelligence Artificielle
Utilisation de l’IA pour assister les utilisateurs.

## Développement durable
Réduction des déplacements et du papier.
