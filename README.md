# SecondVoice

## Overview
**EN:** SecondVoice is an academic web platform designed to support people who face digital access difficulties. It provides human-centered assistance to help users complete online procedures more easily.  
**FR:** SecondVoice est une plateforme web academique concue pour accompagner les personnes en difficulte avec le numerique. Elle propose une assistance humaine pour faciliter les demarches en ligne.

**EN:** This project was developed at **Esprit School of Engineering - Tunisia** during the academic year **2025-2026**.  
**FR:** Ce projet a ete developpe a **Esprit School of Engineering - Tunisia** durant l'annee universitaire **2025-2026**.

## Architecture
**EN:** The repository is organized into two main modules:  
**FR:** Le depot est organise en deux modules principaux:
- `frontend/`: **EN:** user-facing interfaces (chatbot, voice, images, community, profile, settings, subscription) | **FR:** interfaces utilisateur (chatbot, voix, images, communaute, profil, parametres, abonnement)
- `backend/`: **EN:** service/informational pages and base web assets | **FR:** pages de service/information et ressources web de base

**EN:** Both modules use static web technologies and can be served by any HTTP server.  
**FR:** Les deux modules utilisent des technologies web statiques et peuvent etre servis par tout serveur HTTP.
